# Проект: Место

### Обзор

Сайт, где вы можете размещать фото мест, которые посещали, и делиться с друзьями своими путешествиями.
https://maiiapo.github.io/mesto/

### Используемые технологии
* HTML
* CSS (flex, grid, адаптивная вёрстка)
* JavaScript
* BEM
* Figma
